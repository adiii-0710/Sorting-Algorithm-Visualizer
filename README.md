## Overview

Sorting is a fundamental concept in **Data Structures and Algorithms**.  
Understanding sorting algorithms through theory alone can be difficult for beginners.

This project provides a **visual representation of sorting algorithms** using animated bars where each bar represents a value in the array.  
Users can observe how algorithms compare and swap elements step-by-step.

---

## Features

- Generate a **random array**
- Adjustable **array size**
- Adjustable **sorting speed**
- Visual step-by-step sorting
- Clean and interactive user interface

---

## Algorithms Implemented

### Bubble Sort
Bubble Sort compares **adjacent elements** and swaps them if they are in the wrong order.  
The process continues until the entire array becomes sorted.

### Selection Sort
Selection Sort repeatedly finds the **minimum element** from the unsorted part of the array and places it in the correct position.

---

## System Flow

1. User enters the **number of elements**.
2. The system generates a **random array**.
3. Each number is displayed as a **vertical bar**.
4. User selects a **sorting algorithm**.
5. The algorithm sorts the bars **step-by-step visually**.
